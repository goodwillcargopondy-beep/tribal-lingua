# Project Memory

## Core
- Tribal Lingua PWA: Teaches Santhali, Gondi, and Kurukh.
- Tech Stack: React, Vite, Tailwind CSS, Supabase, Lovable AI Gateway.
- Theme: Forest & Earth. Forest green (#2D4739), terracotta (#E2725B), cream (#FDFBF7).
- UX Constraints: High-touch accessibility; min button height 65px, 20px vertical margins.
- Main Nav: 6-tab bottom bar: Home, Learn, Stories, Clan (center enlarged), Quiz, Profile.
- Interactivity: Double-tap vocabulary/phrases to mark learned and gain +5 XP with sound effects.
- Clan UI: Foggy tribal village map that clears based on learning progress percentage.
- Elder Chamber: AI-powered Glyph-Guard via edge function, speaks in tribal language with subtitles.
